# 小程序基础模板

#### 介绍
小程序开发技术模板, 使用 gulp 优化开发步骤

#### 软件架构
微信小程序原生开发
##### 支持
- less
- typescript
- npm 包

#### 使用教程

##### 开发
- 安装依赖并启动
- 打开微信开发者工具, 项目目录选择生成的 dist 文件夹
- 开始开发

##### 发布
- 执行
```
$ pnpm build
```
- 微信开发者工具上传

##### 如何安装依赖

```
$ pnpm i
```

##### 测试环境

```
$ pnpm start
```

##### 生产环境

```
$ pnpm build
```

##### 使用 npm 包
- 直接在根目录安装即可
